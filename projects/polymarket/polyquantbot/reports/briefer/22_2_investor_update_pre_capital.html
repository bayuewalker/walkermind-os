Known Limitations (Non-Critical)


                    

                        
PriceFeedHandler Not Yet Wired to main.py

                        
The WebSocket → PaperPositionManager price bridge module is built and ready. Manual wiring step required to start the background asyncio task. Tick-level mark-to-market via market_prices is already active.

                    

                    

                        
Ledger Auto-Persist Not Yet Automatic

                        
TradeLedger.persist_entry() is available but requires explicit caller invocation. Avoids circular DB dependency in TradeLedger. Full ledger DB sync requires manual call after each record().

                    

                    

                        
SENTINEL Pre-Capital Gate Pending

                        
Full SENTINEL go-live validation (score ≥ 85 required for APPROVED verdict) has not yet been run for Phase 22.2. This gate must be completed before any live capital is deployed.
